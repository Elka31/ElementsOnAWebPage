Elements on a Web Page

Positioning of Elements on a Web Page (definitons)
Description Use case Example
Relative Positioned relative to its normal flow Move element slightly without removing it from flow top: 10px shifts from normal spot
Absolute Positioned relative to nearest positioned ancestor Precisely place in container top: 0 inside parent
Fixed Positioned relative to the viewport Stick element (e.g., nav bar) to viewport top: 0 stays on screen when scrolling
Sticky Switches from relative to fixed on scroll Header that sticks at top while scrolling Header sticks once it reaches top of view
